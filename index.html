Intelligent Excursions
Quicktrips

A website to help you choose your next travel destination

Abstract

Intelligent Excursions is an application which generates personalized holiday recommendations based on a user’s interests, requirements and 5 keywords.

We decided to implement the solution in the form of a website using the Flask Python Web Framework.

The recommendations will be generated using external API calls to find the best combination of accommodation and travel options and their suitability will be analysed through the use of IBM’s Watson Discovery and Tone Analyser APIs.

Key features

  • The ability to register for an account
  • The ability to log into your account
  • The ability to select 5 key words
  • The ability to filter recommendations based on preferences
  • The ability to mix and match hotel and travel recommendations to get a personalised trip
