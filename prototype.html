Implementation of finished functionalities

-We have made a lot of progress in implementing some of the key functionalities of our website so far:

The front end was developed using HTML, CSS and JS through the use of Bootstrap templates. The use of these templates significantly sped up our development process and helped us streamline the design of all our pages.

After some research into web development we decided to use Pythonâ€™s Flask Web Framework as it allowed for a flexible project layout.

We use the open-source relational database management system MySQL to store our User and Recommendation data. As recommendations are personalised and unique to the user the database has a many-to-one relationship where a user can have multiple recommendations, however each recommendation belongs to a single user.

Flask makes routing between pages very easy and the routes.py class allows variables to be passed between pages easily and allows the same page to be used for multiple specialised purposes such as user profiles or details on specific recommendations. These variables are held in field labels in our HTML files: {{ form..label }}.
